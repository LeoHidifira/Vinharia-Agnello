DESCRIÇÃO DO PROJETO

O projeto consiste em um sistema simples de gerenciamento de vinhos desenvolvido com HTML, CSS e JavaScript, utilizando recursos básicos como variáveis, prompt e alert para interação com o usuário. A aplicação permite cadastrar e visualizar informações sobre vinhos, como nome, tipo e safra, de forma prática e direta no navegador. O objetivo principal é praticar lógica de programação e manipulação de dados, simulando um pequeno controle de estoque de vinhos com interface simples e interativa.
